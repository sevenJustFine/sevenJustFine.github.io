宝塔 mysql 无法远程连接

宝塔下新建一个数据库,使用sqlyog无法远程连接.

第一次尝试解决

  • 权限设置为所有人,就是所有ip都可以连接
  • 宝塔’安全’菜单,放行3306端口
  • 云服务器安全组,防火墙添加规则放行3306端口

windows下使用telnet 监测服务器的端口是否开启

无法连接,所以centos7系统防火墙的问题

再次尝试.ok
